# 🏺 考古学人生模拟器

> “你以为的考古：寻龙点穴，摸金校尉；实际上的考古：吃土刮面，盲审被毙。”

**《考古学人生模拟器》** 是一款纯 HTML5/JavaScript 编写的轻量级网页文字生存游戏。玩家将扮演一名误入“天坑”专业的考古学生，在学识、实践、资金、体力和理智（SAN值）的疯狂拉扯中，体验从本科生到博士的学术炼狱之旅。

👉 **[ https://nova-wood.github.io/archaeology-life/]**

## ✨ 核心特色

* **五大数值，硬核生存：** 学识 (ACA)、实践 (PRAC)、资金 (MONEY)、体力 (PHY)、理智 (SAN)。任何一项失控都可能导致强制退学或转行。
* **真实的学术痛点：** 完美还原大三强制秋季下地实习、硕博方向分流（传统考古 vs 科技考古）、导师 PUA、以及被 Reviewer 2 支配的恐惧。
* **薛定谔的理智系统：** 随着 SAN 值降低，你的角色会开始掉头发、挂黑眼圈，甚至进入“薛定谔的理智”发疯状态。
* **丰富的随机事件：** 探方进水、工地小黄、手铲被偷、民工大姨介绍对象、领队灌酒、网民恶意网暴……考古工地没有新鲜事。
* **多结局分支：** 你的选择和最终数值将决定你是成为国家级考古大佬、地方文保员、还是含泪（x）转行。支持“外专业跨考”开局体验学历歧视链。

## 🎮 如何游玩

本项目为单文件架构，无需任何环境配置：
1. 下载/克隆本仓库。
2. 直接在浏览器中双击打开 `index.html` 即可游玩。
3. 完美适配手机端，随时随地体验吃土日常。

## 🛠️ 技术栈
* **前端：** 原生 HTML / CSS / Vanilla JavaScript
* **UI/UX：** 响应式设计 (Flexbox & Media Queries)，极简像素风配色。

## 🤝 贡献与修改
如果你是真正的考古学子，觉得游戏里的折磨还不够真实，欢迎 Fork 本仓库并提交 PR！
你可以通过修改 `index.html` 中的 `getRandomEvent()` 和 `buildActions()` 函数，轻松添加属于你的工地怨念。

## ⚠️ 警告
本游戏包含大量真实的学术圈痛点，可能会引起在读硕博、尤其是考古/历史学专业学生的轻微 PTSD，请在理智（SAN值）健康的情况下游玩。
